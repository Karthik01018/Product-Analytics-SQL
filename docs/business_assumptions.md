# Business Assumptions

This document outlines the assumptions used to simulate business behavior within the InsightFlow Analytics project.

The purpose of these assumptions is to create realistic user, subscription, engagement, and revenue patterns that resemble a growing SaaS business.

---

# User Acquisition Assumptions

User acquisition is distributed across multiple channels.

| Acquisition Channel | Distribution |
| ------------------- | ------------ |
| Google Search       | 30%          |
| LinkedIn            | 20%          |
| YouTube             | 15%          |
| Referral            | 15%          |
| Direct              | 10%          |
| Product Hunt        | 10%          |

### Assumption

Google Search drives the highest user volume, while LinkedIn and Referral users are generally higher-quality leads.

---

# Trial Conversion Assumptions

Trial conversion rates vary by acquisition channel.

| Acquisition Channel | Trial Conversion Rate |
| ------------------- | --------------------- |
| Referral            | 30%                   |
| LinkedIn            | 25%                   |
| Google Search       | 18%                   |
| Direct              | 15%                   |
| Product Hunt        | 12%                   |
| YouTube             | 8%                    |

### Assumption

Referral and LinkedIn users are more likely to become paying customers than users acquired through YouTube.

---

# AI Feature Adoption Assumptions

AI feature adoption varies by subscription plan.

| Plan Type  | AI Usage Rate |
| ---------- | ------------- |
| Free       | 35%           |
| Pro        | 60%           |
| Team       | 75%           |
| Enterprise | 90%           |

### Assumption

Users on higher-tier plans engage more frequently with AI-powered features.

---

# Customer Churn Assumptions

Churn rates differ across subscription plans.

| Plan Type  | Churn Rate |
| ---------- | ---------- |
| Free       | 40%        |
| Pro        | 20%        |
| Team       | 12%        |
| Enterprise | 5%         |

### Assumption

Enterprise customers are less likely to cancel due to deeper product adoption and organizational usage.

---

# Revenue Assumptions

Subscription plans generate recurring monthly revenue.

| Plan Type  | Monthly Revenue |
| ---------- | --------------- |
| Free       | ₹0              |
| Pro        | ₹499            |
| Team       | ₹1,999          |
| Enterprise | ₹9,999          |

### Assumption

Higher-tier plans contribute a disproportionately larger share of revenue.

---

# Feature Usage Assumptions

Feature usage is not evenly distributed.

| Feature            | Usage Share |
| ------------------ | ----------- |
| Task Manager       | 30%         |
| Notes              | 20%         |
| AI Assistant       | 20%         |
| Team Workspace     | 15%         |
| Document Generator | 10%         |
| File Sharing       | 5%          |

### Assumption

Task management and AI-powered features receive the highest engagement.

# Geographic Assumptions

User distribution is concentrated in key SaaS markets.

| Country        | Approximate Share |
| -------------- | ----------------- |
| India          | 45%               |
| United States  | 20%               |
| United Kingdom | 10%               |
| Germany        | 10%               |
| Australia      | 8%                |
| Singapore      | 7%                |

### Assumption

India contributes the largest user base, while the United States contributes higher average revenue per customer.

---

# Demographic Assumptions

Users are distributed across multiple age groups.

| Age Group | Share |
| --------- | ----- |
| 18-24     | 20%   |
| 25-34     | 40%   |
| 35-44     | 25%   |
| 45-54     | 10%   |
| 55+       | 5%    |

### Assumption

Most users fall within the 25-34 age group, reflecting typical adoption patterns for productivity software.

---

# Engagement Assumptions

User engagement levels influence session duration.

| Engagement Level | Average Session Duration |
| ---------------- | ------------------------ |
| Low              | 3-8 minutes              |
| Medium           | 8-20 minutes             |
| High             | 20-45 minutes            |

### Assumption

Highly engaged users spend significantly more time using the platform and are more likely to convert and retain.
